= has_token

== Installation

In <tt>environment.rb</tt>:

  Rails::Initializer.run do |config|
    config.gem 'laserlemon-has_token', :lib => 'has_token', :source => 'http://gems.github.com'
  end

At your application root, run:

  $ sudo rake gems:install
